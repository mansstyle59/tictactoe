# Tic · Tac · Toe

Jeu Tic-Tac-Toe avancé en React avec 6 règles configurables.

## Jouer en ligne
🔗 [Lien GitHub Pages](https://VOTRE_USERNAME.github.io/tictactoe/)

## Fonctionnalités
- Grille 3×3 ou 4×4
- 6 niveaux de difficulté IA
- Règles : Disparition, Blitz, Bombe, Bouclier, Rotation, Échange
- Profils joueurs avec points et classement
- Best of configurable

## Développement local
```bash
npm install
npm run dev
```
